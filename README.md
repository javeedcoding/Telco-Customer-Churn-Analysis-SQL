# 📊 Telco Customer Churn Analysis (SQL Project)

## 🔍 Objective
Analyze customer churn patterns using SQL to identify high-risk customer segments and revenue behavior.

## 🛠 Tools Used
- PostgreSQL
- SQL (CASE, GROUP BY, Aggregations)
- CSV dataset

---

## 📊 Key Insights

- Overall churn rate: **26.58%**
- Month-to-month contract customers have highest churn (~42%)
- Fiber optic users show high churn (~42%)
- Customers in first 12 months churn the most (~47%)
- Customers who churn pay higher average monthly charges
