# Telco Customer Churn Analysis (SQL Project)

## 🔍 Objective
Analyze customer churn behavior using SQL to identify high-risk customer segments.

## 🛠 Tools Used
- PostgreSQL
- SQL (CASE, GROUP BY, Aggregation)
- Python (Data Cleaning)

## 📊 Key Insights
1. Overall churn rate: 26.58%
2. Month-to-month contracts show highest churn (42.71%)
3. Fiber optic users churn at ~42%
4. Early-stage customers (0-12 months) show highest churn (~47%)
5. Higher monthly charges correlate with higher churn

## 💡 Business Recommendations
- Encourage long-term contracts
- Improve early customer retention
- Review pricing sensitivity
